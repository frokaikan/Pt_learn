根据PtBook，每个Model都分为五步：
Preinitialize :: 静态分析，创建内部角色。只执行一次，包括New
Initialize :: 参数初始化，重置本地状态。可能执行多次
---iterate---
Prefile :: 检测先决条件
Fire :: 读入数据，计算，产生输出
Postfire :: 更新角色状态
---iterate end---
